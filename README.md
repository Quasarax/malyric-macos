## About

MALYRIC is built for vocalists who want a dedicated rehearsal environment, not just another media player.

The app combines core playback control with lyric workflow and voice practice tools so singers can train faster and more consistently.

## Current Stage

MALYRIC is currently in **Alpha Test**.

Core features are usable now, and we are actively polishing performance, UX details, and workflow depth.

## Main Features

- Pitch and tempo control for rehearsal.
- Synced lyrics support with fast navigation.
- Built-in metronome and vocal warmup routines.
- Voice recording for self-review.
- Vocal range analysis tools.
- Clean, focused interface for long practice sessions.
