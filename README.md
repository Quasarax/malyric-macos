# MALYRIC Landing (GitHub Pages)

This folder contains a one-page landing site for MALYRIC.

## Quick publish on GitHub Pages

1. Push this folder content to your Pages branch (for example: `gh-pages` or `main`).
2. In repository settings, enable **GitHub Pages** and select the branch/folder.
3. Put your macOS DMG file at `releases/MusicPlayr.dmg`.
4. Open `script.js` and update `release.url` if you want to use a GitHub Releases direct link.

## Free trial + full access note

The page already mentions:
- Free trial availability.
- Full access purchase by contacting your team.

You can update contact email in `index.html` (`mailto:` link).
